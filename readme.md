# 4.其他

训练的数据会放入到logs里面，模型是基于对抗生成网络pix2pix实现的，实现的效果还是一般般

生成器生成效果比较差，loss到后期反而升高了

解决方法：

调整生成器和判别器的学习率，一般生成器的学习率调高一点，判别器的学习率调低一点；

训练2遍生成器再训练一遍判别器
